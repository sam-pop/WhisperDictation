Features
Built for developers.
🔒

Completely Private

All processing happens on your Mac. No audio is ever sent anywhere. No cloud, no API keys, no telemetry.

⚡

Sub-Second Speed

Metal GPU acceleration on Apple Silicon. Beam search for accuracy, greedy decoding for speed. VAD trims silence automatically.

💻

Works Everywhere

Terminal, VS Code, Xcode, Slack, browsers, email. Text injection via CGEvent works in any app that accepts keyboard input.

📚

500+ Dev Terms

Built-in vocabulary biases Whisper toward API, GraphQL, Kubernetes, PostgreSQL, and hundreds more. Add your own custom terms.

✨

Smart Corrections

Auto-capitalizes, fixes 100+ term casings (api→API), adds punctuation, converts number words to digits. Under 5ms overhead.

⚙

Fully Configurable

Rebind the hotkey, choose your model, toggle corrections, add custom vocabulary, select your mic. Native SwiftUI settings.

Get Started
Up and running in 2 minutes.
1

Download WhisperDictation.dmg from Releases

2

Open the DMG and drag WhisperDictation to Applications

3

If macOS blocks it: xattr -cr /Applications/WhisperDictation.app

4

Launch and grant Microphone + Accessibility permissions

5

Go to Settings > Model and download Small Q5 (181 MB)

Or build from source
# Clone and build
git clone --recurse-submodules https://github.com/sam-pop/WhisperDictation.git
cd WhisperDictation
make whisper && make model && make run
